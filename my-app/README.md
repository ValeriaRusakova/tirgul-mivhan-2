מבקשים ממך לבנות אפליקציה
פשוטה, לניהול מוצרים

יש לנו עמוד מרכזי - עמוד הבית - שמציג גריד של כל המוצרים.
כאשר לוחצים על מוצר, מגיעים לעמוד מוצר, ובו רואים פרטים נוספים על המוצר. 


🎯 מטרת התרגיל
לבנות אפליקציית Product Dashboard המדגימה עבודה עם ניווט, ניהול state גלובלי, קריאות API, Context עבור מצב תצוגה כהה/בהירה.

שימוש ב־DummyJSON כ־API

1️⃣ ניתוב (React Router)
יש ליצור את הראוטים הבאים:
/ – עמוד בית


/products – רשימת מוצרים


/products/:id – עמוד מוצר


/cart – עגלת קניות


* – עמוד 404


דרישות
Layout משותף עם Navbar קבוע


ניווט בין עמודים ללא רענון


שימוש ב־route params בעמוד מוצר



2️⃣ רשימת מוצרים (useEffect + API)
API:
https://dummyjson.com/products


דרישות
שליפת נתונים בעת טעינת העמוד


הצגת מצבי:


טעינה


שגיאה


הצגת נתוני מוצר:


תמונה


שם מוצר


מחיר


דירוג


לחיצה על מוצר תוביל לעמוד הפרטים שלו


מימוש pagination או infinite scroll



3️⃣ עמוד מוצר (Product Details)
API:
https://dummyjson.com/products/:id


דרישות
שליפת מוצר לפי ID מה־URL


הצגת:


תיאור


קטגוריה


מחיר


דירוג


סטטוס מלאי


טיפול ב־ID לא תקין


כפתור “הוסף לעגלה”



4️⃣ ניהול State גלובלי (Redux)
State לניהול:
מוצרים בעגלה


כמות מכל מוצר


סה״כ פריטים


סה״כ מחיר


דרישות
הוספה לעגלה


שינוי כמות מוצר


הסרת מוצר


חישוב אוטומטי של מחיר כולל


שמירת העגלה ב־localStorage


הצגת כמות הפריטים בעגלה ב־Navbar

שינוי מיידי של העיצוב בעת החלפה
